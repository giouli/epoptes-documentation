epoptes.common.config module¶

epoptes.common.config.read_groups(filename)¶

Parse a JSON file and create the appropriate group and client objects.

Return a 2-tuple with a client instances list and a group instances list.

epoptes.common.config.read_ini_file(filename)¶

Return a ConfigParser from the contents of a configuration file.

epoptes.common.config.read_plain_file(filename)¶

Return the whole contents of a plain text file into a string list.

If the file doesn’t exist or isn’t readable, return an empty list.

epoptes.common.config.read_shell_file(filename)¶

Return the variables of a shell-like configuration file in a dict.

If the file doesn’t exist or isn’t readable, return an empty list. Also strip all comments, if any.

epoptes.common.config.save_groups(filename, model)¶

Save the groups and their members from model (gtk.ListStore) in JSON format.

epoptes.common.xdg_dirs module¶

epoptes.common.xdg_dirs.default_dirs(lang)¶

Return the default XDG dirs for lang.

These are read from /etc/xdg/user-dirs.defaults.

epoptes.common.xdg_dirs.expand_home(dic, home)¶
epoptes.common.xdg_dirs.get_xdg_dirs(user)¶

Return a dictionary with the xdg dirs for that user.

If any or all of that user’s xdg dirs aren’t set, return the default xdg dirs based on the default system locale. Raise an error if the user doesn’t exist.

epoptes.common.xdg_dirs.localize_dir(path, lnames)¶

Localize a default XDG dir like e.g. Documents/Pictures.

epoptes.common.xdg_dirs.localized_names(lang)¶

Return a dictionary with the translations of XDG directories for lang.

epoptes.common.xdg_dirs.read_config_file(path)¶

Return the contents of a shell-like file in a string list.

If the file doesn’t exist or isn’t readable, return an empty list. Also strip all comments, if any.

epoptes.common.xdg_dirs.read_vars_file(path)¶

Return a dictionary read from a shell vars file.

epoptes.common.xdg_dirs.system_lang()¶

Return the default system locale which is applied to new users.

It is read from /etc/default/locale.

epoptes.common.xdg_dirs.user_dirs(home)¶

Return the user XDG dirs for the user at “home”.

These are read from ~/.config/user-dirs.dirs.

epoptes.common.xdg_dirs.user_lang(home)¶

Return the user language for XDG directories, if set.
